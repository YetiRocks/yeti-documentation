# Get Early Access to Yeti

Yeti is not publicly available yet.

## What you'll get

**A complete backend from a single schema.** Define your data model in GraphQL. Yeti generates REST APIs, GraphQL endpoints, real-time subscriptions, and authentication. No boilerplate.

**Rust performance without writing Rust.** Single-process Rust architecture with RocksDB storage. Sub-millisecond reads, thousands of concurrent connections, minimal memory footprint.

**Real-time by default.** Every table supports Server-Sent Events and WebSocket subscriptions. Clients receive live updates the moment data changes.

**Built-in auth.** Basic, JWT, and OAuth authentication with role-based access control down to individual fields. Add auth to any app with one line of config.

**Hot-reloading custom logic.** Write a Rust resource for business logic beyond CRUD. Yeti compiles and loads it without server restarts.

**One binary. Zero dependencies.** No runtime to install, no database to manage, no container orchestration. A single binary that runs anywhere.
