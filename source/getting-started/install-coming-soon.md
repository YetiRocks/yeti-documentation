# Get Early Access to Yeti

Yeti isn't publicly available yet — but it will be soon, and it's going to change how you build backends forever.

## What you'll get

**A complete backend from a single schema.** Write your data model in GraphQL and Yeti generates REST APIs, GraphQL endpoints, real-time subscriptions, and authentication — instantly. No boilerplate. No plumbing. Just your product.

**Rust performance without writing Rust.** Yeti runs on a single-process Rust architecture with RocksDB under the hood. Your APIs ship fast by default — sub-millisecond reads, thousands of concurrent connections, tiny memory footprint.

**Real-time built in, not bolted on.** Every table you define automatically supports Server-Sent Events and WebSocket subscriptions. Your clients get live updates the moment data changes.

**Auth that actually works.** Basic, JWT, and OAuth authentication with role-based access control — all the way down to individual fields. Add auth to any app with a single line of config.

**Hot-reloading custom logic.** Need business logic beyond CRUD? Write a Rust resource, and Yeti compiles and loads it on the fly. No server restarts. No deploys. Just save and go.

**One binary. Zero dependencies.** No runtime to install, no database to manage, no container orchestration to wrangle. Yeti is a single binary that runs anywhere.
